estou criando a interface do meu app com GTK usando o Glade, nele eu criei uma janela principal, com o id main_window, dentro dele crie uma stack com o ID : stack e dentro do stack a primeira janela que abre no meu aplicativo é a tela de Login. E nela contem os seguintes itens : uma gtk entry para Login com A id login_usuario, um gtk entry para senha com o ID : login_senha, uma gtklabel para informações com o ID : label_mensagem.
e dois gkt button sendo um de login com ID :botao_login, e outro de sair com o ID : botao_sair. e também tem um gtk check button com o ID : lembrar, nele ao ser checado tem que lembrar o login e senha do usuário.

ao clicar no login se bem sucessido devera abrir o menu principal que tem o ID : view_menu_principal, nele notem vários outros GKT button, porem o que eu preciso que funcione no momento é o gtk button sair com o ID : menu sair, para voltar para tela de login que tem o ID : view_login. voltando ao menu principal tenho outro botão chamado cadastro usuário com o ID : cadastro usuário, ao clicar nele tem que abir a view de cadastro com o ID : view_cadastro_usuario, nele temos dois gtk entry sendo usuário com o ID : cadastro_usuario_entry, e senha com ID : senha_cadastro_entry, e dois gkt button, um tem o id : confirma_cadastro_usuario e o outro ID : cancelar_cadastro_usuario, nesse cancelar tem que voltar para o menu principal.
e ao clicar em confirma_cadastro_usuario as informações tem que ser salva em um arquivo txt com o nome dados.txt, se possível criar esse arquivo txt em UTF-8, precisamos apenas criptografar a senha com uma criptográfia muito simples que apenas mudas os caracteres, e na tela de login tem que fazer a verificação dessa senha e do usuário para permitir o login.
